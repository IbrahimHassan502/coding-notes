terminology

tint

tint is when you add white to the colors

and this is achievable by moving the color point in the color picker horizontally to the right (towards the white)

shading

a shade is when you add black to the color

and you ahcieve that by moving the color point vertically downwards (towards the black)

tone

a tone is when you add gery to the color and it is achieved by moving the color point diagonally in the color picker

color picking tools

HSB color model

it stands for hue saturation brightness

hue is basically the color itself then saturation is how vibrant (pure) or dull (greyish) the color is and birghtness is how bright (whitish) or dim (blackish) the color is

hue varies from 0 to 360 with every number preseneting a different color

where brightness and saturation vary from 0 to 100

so by varying the saturation you are changin the place of the color point in the color picker horizontally, meaning that a 50 saturation and 100 brightness will but the point exactly at the middle top of the picker

where a 50 saturation with 50 brightness will put in the center of the picker

variable colors

why use variable colors? simply as it makes it way easier to modify the color scheme of the design saying you have primary and secondary colors you don't have to go change every element that has the changed color by yourself you just changed once it you're ready to go

how to make a variable color

  1. select any element you want to assign the color to
  2. find the fill in the design bar on the right
  3. click on the four dots on the right of the fill and hit the plus icon
  4. type the name and description of the color as instructed in the placeholder

    notice the plus icon on highlighted by the rose rectangular it lets you add shades or something like that I didn't experment with it yet

how to use variable colors

  1. after selecting the element you want to assign the color to under fill on the design bar on the right click on the color square to pick a color
  2. on the top of the new panel click on libraries then choose your color

make varients of variable color

let's say you have a button with the main color and you want to make a varient of it with a darker color in this case you have to break it's link to the main color otherwise you'll be changing the main color itself

you break the link by clicking the broken chain loop beside the color name

components

when to use components

  • when making an animated element like a spiner
  • when making an interacting element like button with hover state
  • when repeating the same element with changes like pricing cards or navbar links

compenets with repeated elements

it's really important espicially with something like navbar links so when you need to make a change like the color or the casing of the letters you just change the parent component and all of them change instead of you going for one by one (just like classes in css)

how to make a component and varient

lets say we're making a navbar element. You want to make three states

  1. the active state (usually home)
  2. the default one
  3. and the hover state

  • the the best practice that you make a seperate component page

    to make a new page click on the "page-1" on the right side of left bar

    here I renamed it to "final design" but in default it says "page-1"

    once you click on it, it will open a list of all the pages you have click the plus icon to add a page (figma allows 3 pages per project in the free plan)

  • go to your component page, choose the type tool either from the top bar or by hitting "t" on the keyboard, then type the first navbar link you want which is usually "home"
  • select that element and from the top bar hit the diamond-like icon
  • name your component from the right bar under current varient
  • make varient by selecting the component then in the place of the diamond-like icon you will find a plus icon looks like it when you click that it makes you a new varient, make the changes you want

  • then to make the hover state make a third varient with the changes you like then from the right open the prototype

    hit the plus icon from the on click dropdown choose while hovering then hover over the default state varient, a small plus icon will show up drag from the icon then connect to the last varient which is the hover state

    from the instant dropdown menu choose smart animation

  • now to use the component in the design go to the main design page from the left bar choose assests

    then drag the component and drop it in the design

  • to choose between the different variations from the right bar under the name of the component (which in our case called nav-item) click on the dropdown menu on the right of "property 1" and choose the name or the state you want

  • to modify the compenent you don't modify the component instant in the main design you go to the main component in the component page and modify that

    note that in that crash course the instructor made the changes for the three varients one by one, so I'm not sure if that's the only way or there's a way to make changes just once and it applies for the three of them

  • if you want to test the hover animatino now click on the play button on the top right of the top bar
